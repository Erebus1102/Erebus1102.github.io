# 1. 分享一种看待币的思路

# 1.1 比较上市公司与加密货币

  • 商业表现与价格表现: 商业模式中流转的是法币、而股票市场上流转的标的则是股票。 商业层面的表现通常表现为供需关系的变化，不直接受股市影响，但会影响股市价格。加密货币的商业与市场价格没有分开，而是一个整体，两者互相影响。
  • 币作为支付与价值:
    • 在币本位定价中，价格的预期变化会影响买卖双方的参与意愿。
      • 当币价预期上涨时，买方会觉得自己把要涨的币支付出去，自己亏了，此时买方继续参与到买卖中的意愿会降低一些。
      • 当币价预期下跌时，卖方会觉得自己收到的币跌了，自己亏了，此时卖方继续参与到买卖中的意愿会降低一些。
    • 在法币本位定价中，币价的预期变化也会影响双方的意愿。
      • 当币价预期上涨时，卖方会觉得自己收到的币数量少了，自己亏了，此时卖方继续参与到买卖中的意愿会降低一些。
      • 当币价预期下跌时，买方会觉得自己支付的币的数量太多了，自己亏了，此时买方继续参与到买卖中的意愿会降低一些 币价的涨跌不论是采用哪种定价，都会使其中一方参与商业活动的意愿降低。所以，对于单个币种来说，如果用途只是支付场景，那么想要更多人将其用于支付（即商业运转的更好）与想要让币价升值，至少单从经济模型上来说，两者是相悖的

# 1.2 看待币的思路转变

  • 由于币既代表了商业上的表现，也体现了股市上实际的价格表现。因此不妨把项目本身的发展看作是商业活动，把币在二级市场的表现看作是金融标的，而经济模型则是决定商业活动与金融标的属性之间的相关性
  • 项目价值的判断：
    • 它是否能解决一些刚需问题，这主要体现项目的可行性
    • 项目本身如何盈利，这主要体现项目的商业逻辑是否成立，并且能成功经营
    • 项目的 Token 与经营是否是比较强的挂钩的，也就是商业发展如何反映到币价上，又或者说是币的价值捕获如何从商业运行上体现出来 目前看来，许多项目刚走完第一步并停留在第二步

# 2. 公链的增值逻辑

# 2.1 公链经济模型中常见的机制设计

# 公链经济模型的机制介绍

  • 分发方式: PoW 和 PoS 是两种主要的分发方式
    • 资源型的 PoW 也分为全部的币通过 PoW 机制产出，以及部分（预挖）通过 PoW 机制产出。
    • 权益型的 PoS，一般会有初始发行量，再在这个基础上进行增发。
  • 使用场景: 包括作为资产、支付 gas 费、投票权、参与 Staking 等，每种使用场景都对公链币的需求产生影响。
  • 回收机制:

波卡特有的国库机制

  • 机制内容： 将增发的一部分，gas 费的一部分，节点 slash 的部分等汇聚到一个资金池中，这个资金池就是国库，国库的 1%会在固定时间内，燃烧掉 1%。这个国库的资金通过链上治理的方式，被用于波卡的生态建设。
  • 机制价值：可以保证只要公链还在正常运行，就会一直有资金可以用于生态建设，波卡也拥有持续且丰富的资金建设生态，这也是波卡相比于其他公链来说，还能持续高频地推出高奖金的黑客松的原因
  • 燃烧机制

以太坊的燃烧机制

  • 机制内容： 2021 年 8 月 5 日，以太坊的伦敦硬分叉升级中激活了 EIP-1559，这改变了以太坊手续费的计算方式，将 gas 费分为了两部分，一部分是通过算法设定的“基本费用”，这部分的 ETH 将会被燃烧掉，另一部分是给矿工的“小费”，作为奖励送给矿工，以激励他们将交易纳入更高的优先级。
  • **机制价值：**自此，以太坊从肯定会通胀的公链，变成会因为生态中的一些应用活跃度，而燃烧掉大量 ETH，进而可能会通缩的公链。

Flow 用手续费填补 Staking 奖励且有燃烧设计

  • 机制内容： 每年 Flow 链的节点总奖励当前设定为初始发行量的 3.75%，并且由固定奖励和交易费奖励两个部分构成固定奖励来源于 FLOW 通证的增发。
    • 当 Flow 链发展初期时交易奖励费很少，固定奖励也就是增发的部分就比较多。
    • 而随着 Flow 链上的生态逐渐发展 Flow 链上的交易费会随之增加，固定奖励的部分就变少了，甚至会出现交易费已经超过发行量的 3.75%的情况，此时固定奖励为 0，FLOW 也就不用增发了，而交易费超过发行量的 3.75%的部分会被销毁。
  • 机制价值：Flow 链虽然是 PoS 共识机制，通过设计机制，既能激励节点，又能严格控制 Flow 链的通货膨胀。

# 公链经济模型的设计思路

常见的公链经济模型，主要有三个可以进行设计的部分：

  • 一是初始发行量部分，比如初始发行 10 亿枚币，其中多少可以是属于团队、投资人、社区或者供后续挖矿所得，当然也可以是全部都通过挖的方式获得。
  • 二是增发部分，（非必须）但是有了增发，可以有多的钱用于做其他重要的事，如 > - 1. 作为国库的一部分，激励生态发展 > - 2. 加上燃烧机制，以免项目通胀过高 增发出来的币可以考虑激励新场景, 就好比互联网公司通过烧钱来抢夺用户一样。
  • 三是交易 gas 费的部分，可在这部分，对于 Token 进行回收，并可以采取分配给矿工、国库或者销毁等措施 一种理想的公链经济模型

# 比较合适的一种经济模型的设计：既有增发，又有国库，同时可以通过燃烧来控制通胀

  • 增发用于激励支持网络正常运行的人，国库用于建设生态，燃烧机制控制通胀可以抵消一部分由通胀带来的心理上的抛盘预期，并且可以实现生态发展的越好，项目如果通缩，进一步提高稀缺性。

# 2.2 回顾公链的特殊事件

# ICO 对 ETH 的影响:

定义：是指一些区块链项目通过发行自己的通证（Token），向投资者募集虚拟货币（如比特币、以太坊）的融资活动。 价值： 由于当时许多项目，是通过募集以太坊的形式参与的。造成了一定时间内大量的 ETH 买盘，而用于参与 ICO 的 ETH 则会被项目方更长期的持有，短时间内 ETH 的买盘大于卖盘，助推了 ETH 从2000人民币上涨到一万以上

# DeFi Summer 对 ETH 的影响:

定义： 由 Compound 推出流动性激励开始，2020年 DeFi 全面爆发，而 ETH 则被广泛运用到各类 DeFi 中，充当其中最可信的背书资产。 价值： ETH 在公链币层面（类似于黄金）从工业属性，转变为金融属性。 ETH 也更多的从使用以太坊网络所需要支付的资源费用，转变为以太坊网络上最具可信的资产，并被沉淀在各类 DeFi 中。也造成了一定时期内，ETH 的买盘旺盛。

# 新公链探索 :

# 1. 币安交易所与 BSC 公链生态双赢的良性循环：

币安在 2020 年 4 月创建了币安智能链（BSC）并于同年 9 月正式上线智能链。BSC 的设计思路包括：

  • 保障高性能低转账费率
  • 兼容以太坊虚拟机 EVM，可快速接入以太坊生态。 而 BSC 的快速崛起，离不开作为全球第一大加密货币交易所的币安为其赋能。
  1. 对于赋能一条公链来说，交易所可以通过在一级市场投资这条公链上的项目或者应用并将这些项目上线到自己的交易所上，并通过市场活动、自身用户以及资金流为项目赋能，并从二级市场成功退出后，再将新的资金继续投资到这条公链上的其他项目上。
  2. 这些项目如果本身的表现也撑得起这些用户和资金的进入，并且发展得更好也会反过来为交易所带来更多的用户和资金，甚至作为一个优秀案例为交易所背书。

# 2. Polygon 公链激励计划

Polygon 在2021年4月，战略从扶持自己的生态转为引入了 ETH 上知名的 DeFi 借贷项目 Aave，并联合 Aave 推出了公链激励计划，使得 Polygon 迅速获得了巨大的成功。

  • 计划内容： 2021 年 4 月 14 日，Polygon 就宣布为借贷协议 Aave 提供流动性挖矿奖励，奖励为预留的总供应量 1%的 MATIC（约 4000 万美元）。随后 Polygon 的 TVL 迅速攀升。激励方案的实践成功，随后又为 Sushiswap、Curve 等知名 DeFi 项目提供了流动性挖矿奖励。
  • 计划价值： Polygon 的锁仓量在短短的一个多月的时间迅速突破 50 亿美元，大涨 30 多倍
  • 激励计划成功的隐性条件：激励计划的螺旋上升在于当时是处于大牛市的环境，所有参与的资金情绪是比较贪婪的。如果，此时是熊市环境，此时的情况就会是螺旋向下的。因此，公链激励计划是一个要看大环境，需要慎用的方法。

# 3. Flow 的 Web2 引流问题

Flow 链上的 NBA 球星卡 NFT 取得了巨大成功，但由于在使用过程中完全不需要用到 Flow 的币、也不会将其资产用到其他链上得到 Defi 之中，因而 Flow 链上的 Defi 发展很慢

  • Web 2 与 Web 3 的用户很难轻易的转化：Web 2 用户尽管进入 Web 3 的生态中，也很难去尝试 Web 3 的可组合性，让自己的资产与各种应用交互。
  • ** 生态发展的好不好，与公链的关系并不一定有关，而取决于生态项目的类别**。目前来看，DeFi类项目与公链的发展是息息相关的，但是 NFT、链游类型的项目，其经济模型通常可以自洽，并且这些项目即使发展的很好，也只会让更多的人买入公链作为 gas 费，与 DeFi 为公链带来的赋能是完全不一样的，从 Axie Infinity 火爆并未对以太坊带来什么反馈，就能感受到。

# 2.3. 影响公链币的涨跌的因素

    1. 经济模型:
    • 锁仓释放: 锁仓减少流通供应，而释放增加供应，直接影响市场供求关系。
    • 通胀机制: 高通胀率可能导致价值稀释，而低通胀或固定供应可能增强稀缺性。
    • 燃烧机制: 燃烧减少流通供应，提高稀缺性，通常视为价值增长的信号。
    • Staking机制: 鼓励长期持有，减少市场流通供应，可能增加价格稳定性。
    • Token的赋能设计: 如投票权、收益权等赋能可以增加持有的吸引力。
  1. 技术迭代: 技术的更新会影响发展或者叙事的情况，如以太坊2.0。
  2. 叙事更新: 一般是指发展方向的调整或者在原有的基础上进行拓展
    • 如 Aster，从 DApp 平台转为日本第一大 Layer1
  3. 重大宣发合作: Web 3/Web 2
  4. 官方运营计划: 官方主动推出的一些运营手段
    • 激励计划
  5. 生态反哺: 生态自发形成而不是官方主动赋能
    • 如 Cosmos 针对 ATOM 持仓进行空投
  6. 外部资源支持:
    • 币安对于 BNBChain 的支持，以及此前FTX对于 Solana 的支持
  7. 资本市场的主观影响:
    • 如韩国交易所上币带来的新流量。
    • 投资人集中解锁对市场带来的冲击。
  8. 宏观经济影响:
    • 市场大环境: 如全球经济形势、监管政策变化、主流金融市场动态等都可能间接或直接影响公链币的价格。

# 2.4. 应用类项目与公链项目的相关性

DePIN，（Decentralized Physical Infrastructure networks）去中心化物理基础设施 DePIN 包括去中心化的存储、计算、传输网络等。逻辑与公链类似，甚至还有新增的部分

  • DePIN 跟公链比，额外的资源与最大的困难都在于如何经营好自己的资源网络
  • DePIN 可以构建资源市场，并且让这些网络资源链接到所有区块链项目，会多一些可以发展的方向和拓展生态以及合作伙伴的方式。

应用链的逻辑也可以借鉴公链的发展思路：应用如果想要发展自己的生态、捕获生态的价值，那么构建一条应用链就是一个可行的方案。

# 3. 应用币的增值逻辑

# 3.1 应用币的商业模式与经济模型

应用币：基于链构建的应用类项目的 Token，这些应用可以是 DeFi/链游等，只要它是基于区块链智能合约构建的应用就属于这类。（单独一条 DAppChain 的情况除外）

公链币由于有主权性、其增值逻辑相对简单:

  • 生态应用通过商业模式塑造出对公链币的新需求，实现生态应用在价值捕获层面反哺公链币
  • 公链币可以在链的层面，对 gas 费进行进一步的设计，实现对币的回收和再分配

应用币因为不同类型的项目其商业模式、经济模型都不相同，情况比较复杂： 最主要的影响币价的两个因素：

  • 供需关系
  • 币的预期（交易市场主要是看预期，或者说是交易者的情绪） 针对项目的所有行为和设计，逐个分析，看看他们是否会影响上述两个因素。

这些行为和设计通常包括但不限于：

  1. 经济模型:
    • 通货膨胀和燃烧机制：
      • 如果通货膨胀比例比较高，可能会引发持续的抛盘
      • 如果开启燃烧机制或者回购销毁，可能会被解读为利好
    • 活动激励：
      • 这个措施主要是币的分发，如果没有后续的使用场景，那么会出现持续的抛盘
  2. 经营数据:
    • 用户数、活跃地址数、交易数、TVL:
      • 如果 Token 和项目经营的关系耦合不强，但也会影响投资者情绪，经营数据给到 Token 的反馈就比较小。
      • 如果 Token 和项目经营的关系耦合强
        • 可以通过经营数据算出营收，如果持有 Token 可以获得分红，那么可以反估 Token 的价值，并且币价不会偏离估价太多。
        • 如果项目有比较自洽的内部循环，此时只需要关注对于这个整体资金流入流出的情况，*一般来说这样的项目都会有类似庞氏的结构，所以当流入资金小于流出资金的时刻，就意味着这个项目的经营情况短时间内就在往下走了，此时就该采取一定的措施了。

比如 Uniswap 自身是获得交易费用作为营收的，他的经营数据与其 TVL 白线优秀。但这些都与 UNI 币并没有直接的联系，所以即使 Uniswap 的经营数据很好也很难让 UNI 获得很大的增长

  1. 商业市场情况:
    • 定义：商业市场是指项目所处的赛道整个市场空间大小与市场格局发展情况
    • 如果项目赛道是有预期的增长空间:
      • 所有人都会最开始受益
      • 会有项目在这个过程中更能捕获增量市场，从而扩大市场份额获得优势，这可能会进一步反映在币价上

Arbitrum 在空投后上交易所，大家对 Arbitrum 的估值计算就是根据它的竞品 Optimism 来计算

  • Step 1: 对比两者的 TVL 可以大概估算到它的估值
  • Step 2: 再进一步估算其比较合理的价格
  1. 商务表现:
    • 商务合作对币价的提升，主要体现在是否为项目带来了增量的市场，我们之前提到的公链与 DeFi 联合推出激励向的合作，就属于是这种直接为项目引流
    • 商务合作，没有直接的体现出市场份额上的增长，但是却带来情绪上的 FOMO
  2. 交易市场表现:
    1. 筹码分布：如果筹码分布太去中心化，并且流通量占总供应量的占比过大，那么，这样的项目要拉升的难度会更高
    2. 币的流动性：流动性不高的项目波动会更大。期货市场的表现，多空比可能会是变盘的因素。
    3. 项目方之外的因素：有些项目方是专门找了市值团队配合做市的，也有一些项目方则没有这样的角色，而是在交易市场中任凭游资自发地组织成为一个或多个庄家。
  3. 宏观市场表现:
    • 主要涉及到 (主要是美国的) 经济数据、财政政策、监管政策
    • 以上这些因子，通常不是单独存在的，通常是同一时间有多个因子同时存在，不同的因子在不同的环境下产生的力量的方向大小、持续时间是不同的。

# 3.2 总结与反思

  • 商业即使在正常发展也会受到币价波动的影响：不论是 DeFi、链游、NFT 在本身经营正常的情况下，会很受牛熊的影响。这有悖于正常的发展逻辑，因为商业正常运转本应该不是受到币价波动的影响的
  • 怎样的设计能够实现商业正常发展与币价波动之间能够解耦，而又可以实现商业的表现可以正确反映到价格上
