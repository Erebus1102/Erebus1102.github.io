# 1. 稳定币投资的收益类型

# 1.1. 利息支付、交易费用和跨链费用

# 利息支付

  • DeFi 借贷平台: 稳定币作为抵押品存放在 DeFi 借贷平台上。
  • 收益来源: 存款人通过借款人支付的利息，获得他们贷出的加密资产的收益。
  • 例子: 包括 AAVE、Notional Finance 和 TrueFi。

# 交易费用

  • 去中心化交易所存款: 稳定币也可以存入去中心化交易所（DEXes）或自动做市商（AMM）。
  • LPs 收益: 存款人或流动性提供者（LPs）赚取交易费用，这些费用由在特定交易对中进行交易的交易者支付。
  • DEXes 例子: 包括 Uniswap、Sushiswap、Curve Finance 和 Balancer。

# 跨链费用

  • 跨链桥接: 稳定币可以存入跨链桥接平台，以便在不同的区块链之间进行交易。
  • 流动性和收益: 跨链桥接平台促进了不同区块链之间的跨链流动性，存款人将赚取由用户支付的跨链费用。
  • 跨链桥接协议例子: 包括 Multi-chain、cBridge 和 Synapse。

# 1.2. 资金成本、衍生品交易对手或期权承销以及保险费用

# 资金成本

  • 期货头寸成本: 持有加密货币的多头或空头期货头寸会产生资金费率成本。
  • 费用支付: 当代币期货的结算价高于其现货价格时，多头持仓者需要向空头持仓者支付费用。

# 衍生品交易对手或期权承销

  • 带杠杆的衍生品交易: 有些 DeFi 平台提供带杠杆的衍生品交易或期权买卖服务。
  • 稳定币作为资本: 存款人将稳定币投入资金池成为衍生品交易者的被动对手方，或作为期权买方的期权承销商。
  • DeFi 协议例子: GMX、Opyn、Ribbon 和 Lyra。

# Example: GMX

  • GMX: GMX是一个允许用户交易加密货币现货和永续合约的去中心化平台。
  1. 流动性提供
    • GLP代币: 存款人可以通过铸造GLP代币来提供流动性。这些资产被添加到包括稳定币在内的多资产池中。
    • 收益: 存款人或流动性提供者获得平台生成的所有交易费用中的70%作为收益。
    • 对手风险: GLP池作为交易者的对手方。这意味着当交易者亏损时，GLP代币持有人将获利，反之亦然。
  2. 资产构成和权重
    • 稳定币权重: 目前，GLP 指数的构成主要由稳定币占据，其总权重为48%。这体现了稳定币在提供流动性和稳定交易平台中的重要性。

# 保险费用

  • DeFi 风险承担: 稳定币投资的另一种收入类型是，存款人将稳定币或其他代币集中起来，以承担 DeFi 中的某些风险，从而赚取保险费。
  • 保险提供者收益: 通过在去中心化资本池中锁定他们的稳定币或其他代币，成为该平台上的保险提供者，根据提供的资本量获得保费的份额作为奖励。
  • 保险 DeFi 协议例子: InsurAce。

# 2. 稳定币资产管理行业

# 1. 普华永道报告

# 1.1. 加密基金管理行业发展

  • 行业增长: 随着法币支持的稳定币数量的增加而呈指数增长。
  • 策略采用: 大多数加密对冲基金正在采用稳定币或 Delta 中性策略。
  • DeFi 利用: 41.6%的加密对冲基金使用了去中心化金融（DeFi）平台以提高收益。
  • DeFi 投资: 78%的加密对冲基金已投资到 DeFi 领域。

# 1.2. 对冲基金特点和趋势

  • 规模: 与传统对冲基金相比，加密对冲基金规模较小（中位数为 2450 万美元）。
  • 资产增长: 2021 年，加密对冲基金管理下的资产增长了三倍。
  • 投资者风险敞口: 投资者票面大小的中位数为 50 万美元，平均值为 150 万美元。
  • 费用结构: 加密基金通常遵循 2-20 的费用结构（2%的管理费和 20%的业绩费）。
  • 团队规模: 与管理下的资产规模相符，加密对冲基金的团队规模也比传统基金要小。
  • 传统对冲基金参与度: 目前 38%的传统对冲基金正在投资加密货币。

# 1.3. 投资领域

  • 前三投资领域: 价值储藏（86%）、去中心化金融（DeFi, 78%）和基础设施（74%）。
  • 权益证明参与: 超过 40%的加密对冲基金参与了权益证明、借贷和借款活动。
  • 稳定币偏好: USDC 和 USDT 是加密对冲基金最青睐的两种稳定币。

# 2. 金融机构对加密货币的参与

  • 黑石集团: 投资比特币和比特币矿业公司，与 Coinbase 建立合作关系。
  • 富达: 在其退休金产品中增加了比特币选项，推出了面向零售客户的加密产品。
  • 查尔斯·施瓦布: 推出具有加密货币曝露的 ETF。
  • 纽约梅隆银行: 成立数字资产部门，将加密货币纳入管理资产。
  • 摩根士丹利: 提供比特币基金，成立数字资产研究团队。
  • 高盛: 准备推出加密投资服务，推出“Datonomy”提高市场透明度。
  • 花旗银行: 推出提供加密投资服务的数字资产部门。
  • 欧洲投资银行: 在公共区块链上发行数字债券。
  • 法国兴业银行: 计划从 MakerDAO 借入 DAI，为资产管理者推出开发加密基金的新服务。

# 3. 稳定币和货币基金及债券的对比

# 3.1. 传统金融与去中心化金融

背景

  • 2022 年美联储利率调整: 导致传统低风险投资如政府和企业债券产生更高的收益。
  • 商业银行动作: 银行接收存款购买国库券（T-bills）作为安全投资。
  • DeFi 市场关联: DeFi 市场与传统市场密切相关，包括直接投资美债和资金的流向。

# 3.2. 稳定币与货币市场工具的比较

  • 流动性产品: 设计上类似货币市场产品，提供流动性。
  • 收益率: 通常由于涉及技术风险、协议风险、监管风险等，稳定币投资的收益率会高于货币市场产品。

# 3.3. 稳定币与债券的比较

  • 固定收益特性: 稳定币和债券都具有固定收益的特性，但风险特性不同。
  • 风险对比:
    • 稳定币: 通常有良好的抵押，没有像债券那样的借款人违约风险，但存在技术风险、监管风险等。
    • 债券: 债券投资的风险包括违约风险、利率风险等。
    • 稳定币投资: 被视为“另类货币市场投资”。

# 4. 行业法规

# 4.1. 加密货币法规的现状

# 全球概况

[! Tip]

  • 除了中国, 大多数大型经济体接受加密货币作为更大金融行业的一部分
  • 各国对于监管加密货币行业采取不同方法
  • 在大多数司法管辖区内，投资稳定币和加密货币是合法的
  • 加密货币和稳定币需求持续增加，促使新的加密货币初创公司进入市场
  • 加密货币市场成为全球投资格局的重要组成部分，监管当局正加强关注

# 美国

  • 2022 年，美国发布新框架，开启加密市场进一步监管。
  • 拜登政府关注非法加密货币活动。
  • 新框架强调修改法律以涵盖数字资产服务提供商，包括数字资产交易所和 NFT 平台。
  • 财政部于 2023 年 2 月底前完成对去中心化金融的非法金融风险评估，7 月前完成对 NFT 的评估。
  • 美联储官员讨论了稳定币的系统性风险，特别是在 2022 年 Terra 稳定币崩盘后。
  • 提出央行数字货币（CBDC）或美元的数字形式可能带来显著好处。

# 欧盟 (EU)

  • 加密货币在 EU 大多数国家合法，但交易所管理由各国决定。
  • 税收政策因国而异，范围从 0%至 50%。
  • 2022 年 3 月，EU 通过了“加密资产市场条例”（MiCA），旨在增强消费者保护、建立明确行业准则，并引入新的许可要求。

# 中国

  • 加密货币归类为财产，用于确定继承。
  • 中国人民银行禁止加密货币交易所运营。
  • 2021 年 5 月，对比特币挖矿实施禁令，导致许多矿工关闭运营或迁移。
  • 2021 年 9 月，加密货币被全面禁止。
  • 中国正在开发数字人民币（e-CNY），并推出中央银行数字货币试点计划。

# 日本

  • 承认加密货币为合法财产。
  • 交易所需在金融服务局注册，遵守 AML/CFT 义务。
  • 成立日本虚拟货币交易所协会（JVCEA）。
  • 加密交易产生的收益视为杂项收入，正在讨论税收和监管方面的工作。

# 英国

  • 加密货币视为财产，非法定货币。
  • 加密货币交易所必须在英国金融行为监管局（FCA）注册。
  • 投资者对加密交易利润需支付资本利得税。
  • 至 2022 年 8 月 30 日，交易所和托管钱包提供商需遵守金融制裁实施办公室的报告义务。
  • 下议院在 2022 年 10 月将加密资产（包括稳定币）视为受监管金融工具。

# 5. 总锁定价值（TVL）

# 5.1. 什么是总锁定价值（TVL）

  • 定义: TVL 衡量存入 DeFi 协议中的所有加密资产的总价值, 并不代表当前有多少贷款未偿还
  • 计算TVL: TVL通常以美元计价，需将存入的代币数量乘以当前的美元价格。如果一个协议接受多种代币的存款，需要分别为每种代币计算TVL，以得出总的TVL

# 5.2. 为什么 TVL 在 DeFi 中重要

  • 衡量标准: TVL是衡量DeFi生态系统大小和活跃度的重要指标。高TVL表明更多资本被锁定在协议中，可能反映了较高的用户信任度和生态系统活跃度
  • 风险评估: TVL可以用来评估生态系统的风险性，因为它提供了一种衡量总体风险价值的方法

# 5.3. 哪个区块链和 DeFi 协议具有最高的 TVL

  • 领先区块链: 以太坊是拥有最大TVL的区块链，达到214亿美元，占总DeFi市场份额的58.3%（截至2023年9月20日）。
  • 其他区块链: Tron的TVL排名第二，仅次于以太坊，总值为66.7亿美元，市场份额为18%（截至2023年9月20日）。
  • 主要协议: DeFi的总TVL中大约有22.87%在Maker协议中。

# 5.4. TVL 指标

  • 市值与 TVL 比率【CoinMarketCap 可查】： 这个比率衡量了在协议中存储的每一美元的价值。计算方式是将市值除以协议的 TVL 以获得该比率。这个比率可以帮助投资者判断 DeFi 协议的原生代币是被低估（便宜）还是被高估（昂贵）。
  • 全摊薄估值对 TVL 比率【CoinGecko 可查】: 该指标衡量了如果所有代币都流通在市场上，那么以今天的代币价格来计算，该协议的总价值是多少。它通过将完全摊薄的市值除以该协议的 TVL 来计算，并考虑了代币的通胀。

# 6. APY, APR,及复利

# 6.1. 年化收益率 (Defi Llama/Exponential Finance 可查)

  • 定义：APY (Annual Percentage Yield) 是投资的年化回报率，考虑到随余额累积的复利。
  • 特点：APY 在加密货币投资平台提供的比传统银行要高得多，由于与加密货币领域相关的风险和利润较大。

当用户在 AAVE 上借出他的$100 USDC 一年，利率为10%，7天 APY，则利率将每7天复合一次，用户一年后会收到$110.51 USDC。

# 2. 年化百分率 (APR)

  • 定义：APR (Annual Percentage Rate) 是投资者可以期望通过其投资赚取的回报率。
  • 特点：与 APY 不同，APR 不包括复利，因此 APY 始终会导致更高的最终金额。

  • 如果投资者在 DeFi 平台上以 10%的 APR 存入 100 美元的 USDC，他将在一年后获得 10 美元 USDC 的利息
  • 但是使用10%的APY，情况发生了显著变化。投资者将能够在一年后赚取更高的利息，即10.12美元

# 3. 复利的影响

  • 定义: 复利是指资本的利息再投资以产生额外利息。
  • 在加密投资中的作用
    • 加密货币平台上的 APY 经常变化，它们通常只是估算值。
    • APY的计算主要取决于加密资产的供求关系和项目的指定区块链协议。

# 复利计算

  • 对于加密投资，复利可能按不同周期计算，如每天、每周或每月。
  • 投资者应了解所投资协议的复利计算方法，因为这会显著影响投资收益。

# 4. 投资APY和APR的其他考虑因素

  • 费用和成本: 投资者需要考虑交易费、提取费和协议运行费等。
  • 变动与固定的APR和APY: 一些协议提供固定利率，而其他可能提供变动利率。
  • 额外福利和奖励: 投资者可能会获得额外的代币奖励或其他福利。
  • 平台的声誉和规模: 投资前应考虑平台的可靠性、安全性和市场声誉。

# 6 . 传统比率在 DeFi 中的对标

# 6.1. 价格-销售额比率 (P/S Ratio)

  • 定义：P/S Ratio: 衡量投资者愿意为每一美元的年化收入支付多少金额。计算方法是市值除以协议的年化收入。 *收入是指通过销毁机制进入协议的财库或直接流向其代币持有人的费用份额
  • DeFi 应用
    • 在 DeFi 中，收入可能来自于各种费用，例如交易费或借贷利息。
      • 例如，去中心化交易所 PancakeSwap 从每笔交易中收取 0.25%的费用。其中 0.17%的费用分给流动性提供者，剩下的 0.08%则进入协议的财库，并用于 SCAKE 的回购和销毁
    • P/S 比率用于衡量 DeFi 代币的价值和市场情绪。一个低的 P/S 比率可能表明该 DeFi 代币相对便宜。重要的是要必须将相似的项目相互比较 (例如 DEX 与 DEX)

# 6.2. 价格-费用比率 (P/F Ratio)

  • 定义：P/F Ratio: 将完全摊薄的市值除以协议的总收入来计算的比率。总收入指用户支付的总费用。
  • 应用：
    • 衡量 DeFi 代币相对于项目总收入的价值。
    • 高比率可能意味着代币被高估，反之亦然。但各个行业之间的比率可能有很大的不同，在对比时只能将其与两个同赛道的代币进行比较
    • 可以直接从 Token Terminal 的网站上获取 P/F 比率数据，以节省不用进行计算的时间

# 6.3. 使用这些比率的原则

  1. 第一步-相同框架比较: 比较不同协议时应确保在相同的财务和经济环境下进行。
  2. 第二步-类似协议比较: 比较时应选择功能和市场相似的协议，确保比率有意义。
  3. 第三步-考虑基本面: 除了比率，还应考虑项目的潜力、市场趋势和其他相关因素。
  4. 第四步-选择潜力项目: 在所有比较和分析后，选择具有较低比率但更高潜力的项目进行投资。

# 7. 稳定币底层资产的品质

# 7.1. 理解底层资产

  • 定义: 底层资产是指支撑稳定币价值的实际资产或资金。
  • DeFi 结构产品: 在 DeFi 中，投资者提供资本给产品，如贷款协议、DEX、期权/衍生品承销、基金/投资组合、跨链桥接等。
    • 贷款协议: 抵押品包括稳定币和其他加密货币。
    • DEX: 包含交易对/交易池中的所有代币。
    • 期权/衍生品承销: 仅涉及稳定币，但存在市场波动风险。
    • 基金/投资组合: 根据策略，范围可能广泛。
    • 跨链桥接: 涉及流动性池中提供的稳定币的所有版本。
  • 案例：
    • Curve 池
    • AAVE 及其分叉项目
    • Gro 协议

# 8 . 协议经济风险

# 8.1. 什么是经济风险

  • 定义: DeFi 协议中的经济风险通常被忽视，它不仅涉及智能合约的安全性和代币经济学，还包括财务可行性。财务可行性至关重要，因为技术创新不能取代基本的财务原则，例如可持续的利率。

# 8.2. 在 Pool 2 中的超高年化收益率

  • 常见现象: 一些协议的年化收益率（APY）超过 1000%，这些高收益通常是不可持续的设计。
  • Pool 2 机制: 指的是 DeFi 平台提供的收益或流动性挖掘机会，通过激励用户提供流动性，即将代币存入智能合约以获取去中心化金融服务。
  • 风险考虑: 这种高收益率可能会引发对池（以及因此而上涨的平台代币价格）的购买兴趣，提高了所有池的年化收益率（APY）。

# 8.3. 去中心化交易所池中的流动性失衡

  • 市场影响: 市场力量的影响，如没有恶意设计的情况下，也可能导致投资者遭受损失。
  • 案例: MIM Curve 池的情况分析，展示了流动性挖矿农民对高回报率的吸引以及经济冲击事件。
  • 影响: 撤出大量流动性、存款人选择撤回资产以避免风险、池组成不平衡导致退出费用升高等。

# 8.4. 高通货膨胀代币模型

  • 设计缺陷: 一些协议存在设计错误，可能在一夜之间导致超级通货膨胀，摧毁投资者的价值。
  • 案例: Iron. Finance 事件，其中 TITAN 代币崩溃，被称为“世界首次大规模的加密银行挤兑”。
  • 后果: IRON 稳定币脱离锚定价，价格源预言机的延迟导致套利无利可图，失去了超过 20 亿美元的总锁定价值（TVL）。

# 结论

  • 财务可行性: 经济风险在 DeFi 中同样重要，财务可行性和代币经济学不可或缺。
  • 风险意识: 投资者应对这些风险有清晰的认识，并在投资决策中加以考虑。
